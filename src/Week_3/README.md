This directory contains hands-on exercises, coding implementations, and mini-projects completed during **Week 3** of the **Cognizant Digital Nurture 5.0 – Java Full Stack Engineer (FSE)** program.

## 📂 Modules Covered

### 🌱 Spring Core and Maven
- Configuring a Basic Spring Application
- Creating and Configuring a Maven Project
- Implementing Dependency Injection (DI)
- Library Management Mini Project

### 🍃 Spring Data JPA with Spring Boot & Hibernate
- Spring Boot Fundamentals
- Spring Data JPA
- Hibernate ORM
- Entity Mapping
- Repository Layer
- CRUD Operations
- Database Integration

## 🎯 Learning Objectives

- Understand the fundamentals of Spring Core and Maven.
- Learn Dependency Injection (DI) and Inversion of Control (IoC).
- Build modular Java applications using Spring.
- Configure Maven projects and manage dependencies.
- Implement persistence using Spring Data JPA and Hibernate.
- Perform CRUD operations with relational databases.
