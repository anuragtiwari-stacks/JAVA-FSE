# Bank PL/SQL Stored Procedures

This repository contains Oracle PL/SQL stored procedures for common banking operations.

## Scenarios

### 1. ProcessMonthlyInterest

Applies a monthly interest rate of 1% to all savings accounts.

### 2. UpdateEmployeeBonus

Updates employee salaries in a specified department using a bonus percentage parameter.

### 3. TransferFunds

Transfers funds between accounts after validating sufficient balance.

## Concepts Used

* Stored Procedures
* Parameters
* UPDATE Statements
* Conditional Logic
* Transactions
* COMMIT / ROLLBACK

## Technology

* Oracle PL/SQL
* Oracle Live SQL
