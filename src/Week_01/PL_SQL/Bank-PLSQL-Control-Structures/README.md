# Bank PL/SQL Control Structures

This repository contains PL/SQL solutions for common banking operations using control structures such as loops, cursors, and conditional statements.

## Scenarios Covered

### Scenario 1: Loan Interest Discount

Apply a 1% discount to loan interest rates for customers whose age is greater than 60 years.

### Scenario 2: VIP Promotion

Promote customers to VIP status by setting the `IsVIP` flag to TRUE when their account balance exceeds $10,000.

### Scenario 3: Loan Due Reminder

Generate reminder messages for customers whose loan due dates fall within the next 30 days.

## Concepts Used

* PL/SQL Blocks
* Cursors
* FOR Loops
* IF Statements
* UPDATE Statements
* DBMS_OUTPUT
* Date Functions

## Technology Stack

* Oracle PL/SQL
* Oracle Live SQL / Oracle SQL Developer
