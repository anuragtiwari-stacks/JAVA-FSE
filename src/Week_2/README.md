# 🧪 Test Driven Development (TDD)

This directory contains practical examples and exercises covering the fundamentals of **Test Driven Development (TDD)** using **JUnit 5**, **Mockito**, and **SLF4J Logging**.

## 📂 Topics Covered

- ✅ Setting Up JUnit
- ✅ JUnit AAA (Arrange – Act – Assert)
- ✅ JUnit Assertions
- ✅ Mocking & Stubbing with Mockito
- ✅ Verifying Interactions
- ✅ SLF4J Logging

## 🎯 Objective

Learn how to write reliable, maintainable, and testable Java applications by following the **Red → Green → Refactor** cycle of Test Driven Development.

## 🛠️ Technologies

- Java
- JUnit 5
- Mockito
- Maven
- SLF4J
