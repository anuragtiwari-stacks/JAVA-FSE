## 📖 About

This repository contains all **hands-on exercises**, **weekly assignments**, **coding challenges**, and **projects** completed as part of the **Cognizant Digital Nurture 5.0 – Java Full Stack Engineer (FSE) Program**.

The objective of this repository is to strengthen enterprise Java development skills through practical implementation of:

- ☕ Core Java
- 🧩 Design Patterns
- 📊 Data Structures & Algorithms
- 🗄️ SQL & PL/SQL
- 🌱 Spring Framework & Spring Boot
- 🧪 Unit Testing (JUnit & Mockito)
- 📦 Maven
- 🌿 Git & GitHub
- ⚙️ CI/CD Fundamentals
- 💼 Enterprise Application Development
